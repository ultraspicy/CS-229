\begin{answer}

Let $\beta = [\beta_1, \beta_2, \cdots, \beta_n]$, similarly we can get the following

\begin{equation*}
	 \theta^{[t]}_i = (1 - \alpha \beta_i) ^ t \theta ^{[0]}, i \in \{1,\dots,d\}
\end{equation*}

and 
\begin{equation*}
    0 < \alpha < min{\frac{2}{\beta_i}} , i \in \{1,\dots,d\}
\end{equation*}

when $t\to\infty$ and GD converges,
\begin{equation*}
    \theta ^ \dagger = 0
\end{equation*}
